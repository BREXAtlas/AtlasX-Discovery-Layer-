# MVP Desktop App Roadmap

The app comes last. The repository should stabilize the CLI engine, schemas,
routing, and outputs before building a desktop or mobile interface.

## Stages

1. Stabilize CLI pipeline.
2. Add source routing.
3. Add flexible source atoms.
4. Add app-ready output contracts.
5. Add local notebook workspace format.
6. Add desktop UI wrapper.
7. Add mobile companion later.

## Near-Term Focus

- Keep the existing CLI working.
- Keep offline deterministic mode working.
- Keep local LLM mode working.
- Keep cloud providers optional.
- Expand source support safely.
- Write stable notebook output files.
