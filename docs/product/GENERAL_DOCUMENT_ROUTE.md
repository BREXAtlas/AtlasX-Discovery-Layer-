# General Document Route

The general route is for ordinary PDFs, articles, essays, reports, notes,
transcripts, and other non-research sources.

General Notebook Mode extracts `SourceAtom` records rather than research-only
`KnowledgeAtom` records.

## Extraction Targets

- Main topic.
- Core claims.
- Supporting points.
- Key terms.
- People and entities.
- Timeline or sequence.
- Assumptions.
- Limitations.
- Contradictions.
- Connections.
- Open questions.
- Action items.
- Study notes.
- Summary.
- User takeaway.

## Safeguards

- Preserve source traces.
- Use `unknown`, `not reported`, or `not applicable` when a field cannot be
  extracted.
- Do not invent details.
- Do not reproduce copyrighted full text.
- Keep human review required.

Learning and strategic brief routes currently use the same SourceAtom extraction
path. Specialized agents can be added later.
