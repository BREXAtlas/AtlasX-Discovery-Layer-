# Ecosystem Connections

AtlasX Discovery Notebook can connect to future AtlasX ecosystem products while
keeping the CLI as the core processing engine.

## AskAtlasX

Conversational, source-grounded Q&A over processed notebooks.

## BREXAtlas / Bioelectric Research Encyclopedia

Domain-specific research adapter and example knowledge base for bioelectric
research.

## Atlas Foundations

Convert sources into structured learning modules.

## Atlas Quest

Convert source insights into gamified quests and learning challenges.

## AtlasX Literacy

Convert sources into research literacy, information literacy, and study skill
outputs.

## AtlasX Discovery Layer CLI

The core processing engine for routing, extraction, graphing, reporting, and
audit trails.

## Future Institutional Research Database Connectors

Library and database integration layers can feed metadata, abstracts, notes, or
lawfully supplied source text into local AtlasX workspaces.
