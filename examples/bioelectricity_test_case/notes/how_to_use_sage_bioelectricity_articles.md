# Using SAGE Bioelectricity Articles With AtlasX

This folder intentionally contains citation metadata only. It does not contain
article text from SAGE.

To use AtlasX with these sources:

1. Verify that you have lawful access through your institution, personal
   subscription, open access status, author permission, or another valid route.
2. Keep any downloaded PDFs or full text in a local folder that is ignored by
   Git.
3. Add only citation metadata, DOI, abstracts, lawful notes, or derived
   source-traceable knowledge atoms to public repositories.
4. Preserve DOI and publisher links so readers can verify the source.
5. Do not use extracted notes to replace the original article or misrepresent
   the authors.
