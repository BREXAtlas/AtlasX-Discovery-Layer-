# General Notebook Project

This example demonstrates AtlasX Discovery Notebook routing for non-research
documents. The source text is synthetic and safe to commit.

Run:

```bash
atlasx run --project examples/general_notebook_project --provider offline --route auto
```

Expected notebook outputs:

```text
outputs/notebook/source_routes.json
outputs/notebook/source_atoms.json
outputs/notebook/source_index.json
outputs/notebook/notebook_summary.md
```
