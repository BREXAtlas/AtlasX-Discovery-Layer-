# Sample Project

This demo uses two fictional, public-domain-style toy paper files. They are not
real articles and do not contain copyrighted text.

Run:

```bash
atlasx run --project examples/sample_project --provider offline
atlasx graph --project examples/sample_project
atlasx report --project examples/sample_project
```

Outputs will be written to `outputs/`, which is ignored by Git.
